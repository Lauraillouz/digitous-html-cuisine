# Digitous - Cooking website (school project)

## Description
* Reproduce a cooking website from a template
* The website is responsive
* Must: use bootstrap classes and avoid as much as possible css

## Stack
* HTML
* CSS
* Bootstrap

## Get started
1. Install npm:

  `npm init -y`

2. Install bootstrap: 

  `npm i bootstrap`

3. Open html index:

  `open index.html`
